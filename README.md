3D Platformer game using Unity
I’m excited to share my platformer game project, where I explored the fundamentals of game development by designing a skill-based experience centered around movement, jumping, and precision. This project helped me understand how different elements like objectives, methodology, mechanics, and player dynamics come together to create an engaging game.

🎯 Objectives:
* Develop a *skill-based platformer* focused on timing and precision

* Create smooth and responsive *player movement mechanics*

* Design an engaging *level environment* with floating platforms

* Introduce *challenge and replayability* through a timer system

Strengthen my understanding of *core game development concepts*

Methodology:
* 📌 Started with a simple concept of navigating across *floating sky platforms*

* 🧩 Planned core systems like movement, jumping, and collision detection

* 🎮 Implemented gameplay mechanics using a game engine

* 🧪 Tested continuously to refine controls and improve player experience

*📈 Optimized performance and ensured smooth gameplay flow

🕹️ Player Movement System:
* 🏃 Smooth *left and right movement* for precise positioning

* 🦘 *Jumping mechanics* as the core gameplay element

* 🎯 *Variable jump height* based on player input

* ⚡ *Momentum-based motion* for realistic and skill-driven control

🧍 Smooth transitions between *idle, movement, and landing*
Core Mechanics:
* ☁️ Navigation across *floating platforms*
⏱️ *Timer-based challenge* to improve performance

* ⚠️ *Failure system* where falling resets progress

* 🎯 Increasing difficulty with platform spacing and placement

📊 Game Dynamics:
* ⚖️ *Risk vs Reward:* Safe vs challenging jumps

* 🧠 *Spatial Awareness:* Judging distance, height, and landing

* 📈 *Progressive Difficulty:* Gradual increase in challenge

🎯 Focus on *precision and timing*
